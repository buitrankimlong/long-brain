# Facebook post KHONG kem link - Meta giam reach bai co external link

## Boi canh
Khi dang bai tu dong len Facebook Fanpage, ban dau code dang bai kem link nguon goc. User yeu cau khong duoc kem link vi Meta phat (giam reach) bai viet co external link.

## Giai phap
Luon dang bai kem anh (post_with_photo) voi message la content, KHONG kem link. Fallback: dang text thuan (post_to_facebook) cung KHONG kem link. Anh lay tu thumbnail cua bai goc (JSON-LD image, og:image, hoac RSS enclosure).

## Duc ket
Khi dang bai tu dong len FB Fanpage: KHONG BAO GIO kem link trong bai. Meta giam reach manh bai co external link. Chi dang anh + text.
