# X/Twitter scraping 2026 — Nitter chết, cần Apify hoặc API trả phí

## Boi canh
Research X/Twitter monitoring. Nitter chỉ còn 3 instances hoạt động, không đáng tin. yt-dlp không hỗ trợ X profiles. RSSHub cần X API key (trả phí). X API free tier ĐÃ BỊ XÓA (tháng 2/2026) — giờ pay-per-use $0.005/read, tối thiểu $50/tháng.

## Giai phap
Apify ($5-10/tháng, free tier 20K tweets/tháng) là lựa chọn tốt nhất cho budget thấp. X API v2 + Tweepy ($50-150/tháng) nếu cần reliability cao. Playwright + GraphQL interception miễn phí nhưng fragile, cần maintenance 2-4h/tháng.

## Duc ket
X/Twitter 2026: KHÔNG CÒN free tier nào đáng tin. Apify = best value $5-10/mo. Playwright = DIY nhưng fragile. Nitter = chết. yt-dlp = không hỗ trợ.
