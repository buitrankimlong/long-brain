# Data-Analyst-Excel-Dashboard-Project

## Mo ta
Dự án phân tích dữ liệu bằng Excel & BI tools: 4 dashboards chuyên nghiệp (Sales Cohort Analysis, Superstore Profitability Geo, HR Analytics, Real Estate Sales). Datasets: online_retail_II (UK commerce), US Superstore (sales, profit, geo analysis). Dashboards sử dụng Pivot Tables, charts, conditional formatting, slicers interactif.

## Stack
- Excel (Advanced)
- Pivot Tables
- Power Query
- Conditional Formatting
- Charts & Graphs

## Quyet dinh quan trong
- Cấu trúc: Raw Data sheet + 2-3 Analysis sheets + Dashboard sheet riêng\n- Cohort Analysis: Customer Journey từ first purchase → repeat rates\n- Geo Analysis: Sales/Profit by State/Region, top performers visual\n- HR Dashboard: Headcount, attrition, department salary ranges\n- Real Estate: Price trends, location heatmap, property types\n- Slicers + Pivot caching để dashboards update dynamic

## Bai hoc rut ra
Excel dashboard best practices: 1) Tách Data → Processing → Viz layers; 2) Pivot Table source cần clean data (no blanks, consistent types); 3) Conditional formatting: color scales, data bars rõ ràng; 4) Chart titles, axis labels rõ ràng; 5) Slicer placement nested logic cho multi-field filter; 6) Large datasets (91MB online_retail_II) → cần Pivot cache optimization.

## Ket qua
4 Excel dashboards hoàn chỉnh với visualizations, análises cohort/geo/HR/realestate, file size lớn (91MB) chứa 541k rows UK commerce data + Superstore US 1st/2nd year metrics.
